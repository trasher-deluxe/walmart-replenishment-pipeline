# Prueba Técnica — Data Scientist

Eres Data Scientist en una cadena de retail con presencia en múltiples regiones de México.

Se te proporciona un conjunto de datos histórico de operaciones de tienda. Tu tarea es construir una solución basada en datos que sea útil para el negocio.

---

## Dataset

Los archivos se encuentran en la carpeta `data/`:

| Archivo | Descripción |
|---------|-------------|
| `transactions.csv` | Transacciones diarias por tienda y categoría de producto |
| `stores.csv` | Información y características de cada tienda |
| `calendar.csv` | Calendario con eventos y variables temporales |

El diccionario de datos completo se encuentra en `data_dictionary.md`.

---

## Tu tarea

No hay instrucciones adicionales. Tú defines:

- Qué problema de negocio resolver
- Cómo abordarlo
- Cómo validar que tu solución funciona
- Qué métricas demuestran su valor

Repositorio **público** en GitHub con tu solución completa.

No hay restricciones de stack, librerías, o enfoque. No hay una única solución correcta.
